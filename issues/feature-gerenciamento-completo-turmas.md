# Feature: Gerenciamento Completo de Turmas (CRUD)

Esta feature implementará todos os endpoints da API necessários para criar, ler, atualizar e deletar Turmas (Classes), bem como gerenciar seus relacionamentos com Alunos (matrículas) e Instrutores (atribuições).

## Tarefas:
- Implementar CRUD Básico para Turmas
- Implementar Gerenciamento de Alunos em Turmas (Matrícula)
- Implementar Atribuição de Instrutores a Turmas
